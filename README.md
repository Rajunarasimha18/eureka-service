Eureka Service
Service Discovery Server for the E-Kart Microservices Architecture built using Spring Boot and Netflix Eureka.

🚀 Tech Stack
Java 21
Spring Boot 3.5.7
Spring Cloud Eureka Server
Maven

📌 Overview
This service acts as the central registry for all microservices in the application.
Services like Product Service, Order Service, and Admin Service register with Eureka for dynamic service discovery and communication.

⚙️ Features
Service Registration & Discovery
Centralized Registry

▶️ Run the Application
mvn spring-boot:run
